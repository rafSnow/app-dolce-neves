# Histórias de Usuário (AS-IS)

Este documento detalha as histórias de usuário implementadas no estado atual do sistema "Doce Neves".

## Épico 1: Gestão de Insumos e Receitas
- **US-01**: Como confeiteira, quero cadastrar meus ingredientes e embalagens com preço e peso, para que eu possa acompanhar meus gastos de estoque.
- **US-02**: Como confeiteira, quero calcular o custo dinâmico do gás utilizando uma calculadora específica de botijão, para que o custo de assamento seja repassado aos bolos.
- **US-03**: Como confeiteira, quero criar a ficha técnica dos meus produtos (ex: "Bolo de Cenoura"), adicionando as gramas exatas que uso de cada ingrediente, para que o sistema me diga quanto custa fabricar o produto.
- **US-04**: Como confeiteira, quero saber rapidamente se estou vendendo meus produtos com lucro ou prejuízo, para que eu ajuste os preços sem fazer cálculos complexos no papel.

## Épico 2: Gestão de Vendas (Pedidos)
- **US-05**: Como confeiteira, quero cadastrar clientes com nome e telefone, para selecioná-los ao gerar novos pedidos.
- **US-06**: Como confeiteira, quero lançar pedidos de venda adicionando os produtos fabricados, para oficializar a encomenda.
- **US-07**: Como confeiteira, quero registrar pagamentos adiantados (Sinal de 50%) e pagamentos na entrega no mesmo pedido, para que eu saiba quem me deve e quanto falta receber.
- **US-08**: Como confeiteira, quero que, ao fechar um pedido, o sistema abata automaticamente do meu inventário os ingredientes usados na receita dos doces vendidos, para não me preocupar com baixa manual.

## Épico 3: Controle Financeiro e Dashboards
- **US-09**: Como gestora do negócio, quero visualizar o fluxo de despesas não ligadas a insumos (ex: Investimentos, Outros), para ter controle sobre a saída total de dinheiro da minha empresa.
- **US-10**: Como gestora do negócio, quero abrir o Dashboard e ver num relance meu "Ticket Médio" e minha "Margem de Lucro Mensal", para entender a saúde financeira geral da confeitaria.
- **US-11**: Como gestora do negócio, quero conseguir exportar todas as listagens para arquivos Excel, para poder mandar para a contabilidade ou imprimir tabelas caso necessite.

## Épico 4: Configurações Gerais
- **US-12**: Como confeiteira, quero definir o nome do meu estabelecimento nas configurações, para que todas as telas exibam a marca da minha empresa.
